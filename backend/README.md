# FastAPI Backend Engine

This directory contains the FastAPI backend for the AI system. It serves as the central hub for receiving data, communicating with the AI Engine, and exposing REST APIs and WebSockets to the frontend.

## Features implemented

- **FastAPI Framework**: High-performance asynchronous REST API.
- **SQLite Database**: SQLAlchemy ORM integrated for storing video metadata and analysis results.
- **REST Endpoints**:
  - `POST /upload`: Accepts video uploads, saves them, and mocks AI engine processing.
  - `GET /vehicle`: Returns a list of uploaded videos and their status.
  - `GET /detections`: Retrieves the latest AI detection results.
  - `GET /history`: Returns historical analysis records.
  - `GET /analytics`: Aggregates detection and upload metrics.
  - `GET /simulation`: Returns mock simulation environment data.
  - `GET /health`: Simple health check endpoint.
- **WebSocket Endpoint**: Real-time communication endpoint at `ws://<host>:<port>/live`.
- **AI Engine Service**: Mock integration with an external AI engine using structured JSON.
- **Middleware**: Request/Response timing and error logging middleware.

## Project Structure

```
backend/
 ├── app/
 │   ├── api/
 │   ├── routes/       # API endpoints (upload, vehicle, detections, etc.)
 │   ├── services/     # Business logic (e.g., ai_engine.py)
 │   ├── models/       # SQLAlchemy models
 │   ├── schemas/      # Pydantic validation schemas
 │   ├── database/     # DB connection and session management
 │   ├── websocket/    # WebSocket handlers
 │   ├── middleware/   # Custom FastAPI middleware (e.g., logging)
 │   ├── config/       # Environment and App Settings
 │   └── utils/        # Helper scripts (file saving)
 ├── uploads/          # Stored video uploads
 ├── logs/             # API logs generated by middleware
 ├── main.py           # Application entry point
 └── requirements.txt  # Python dependencies
```

## Setup & Running

### Requirements
- Python 3.9+

### Installation
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Server
Run the FastAPI application with Uvicorn:
```bash
python main.py
# OR
uvicorn main:app --reload
```
The server will start on `http://127.0.0.1:8000`.

### Documentation
Once the server is running, the interactive API documentation (Swagger UI) is automatically available at:
[http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
